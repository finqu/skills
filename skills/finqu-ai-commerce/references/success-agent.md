# Success Agent

Finqu's built-in AI agent for commerce operations.

## Overview

The Success Agent is an AI assistant that helps merchants manage their store. It can answer questions, execute tasks, and plan complex multi-step operations.

## Modes

### Ask Mode

The agent answers questions using:

- Store data (products, orders, customers)
- Configured guidelines and knowledge
- Connected MCP tools for external data

Example: "What were our top-selling products last month?"

### Agent Mode

The agent actively executes tasks:

- Modify product titles and descriptions
- Update inventory
- Generate marketing content
- Process bulk operations

Example: "Update all summer collection products to show 20% discount."

### Planning Mode

For complex tasks, the agent creates a step-by-step plan:

1. Analyzes the request
2. Breaks it into sequential steps
3. Presents the plan for review
4. Executes after approval

Example: "Prepare the store for the holiday season" → generates a multi-step plan covering inventory, promotions, content updates, etc.
